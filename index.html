LightBird

A tracker for PREPA outages

View project onGitHub

LightBird is the project that powers the http://sefuelaluz.com website which is a tracker for PREPA. The project is written in Python with a MySQL backend. Note that the website is in Spanish since its for a spanish audience, but the code is all English.
